Week 12 Achievements:

1. Synthesis of the 12-Week Study & Research Path:

Over the 12-week timeline of the First Cloud Journey program, all knowledge milestones have been successfully completed:

  • Weeks 1 - 2 (Onboarding & Cost Guardrails): Onboarded with the team, studied fundamental S3/EC2 concepts, and set up financial safety checks using AWS Budgets.
  • Weeks 3 - 4 (Compute & Database Infrastructures): Studied EC2 instance types (Public/Private Subnets), IAM security policies, and RDS database management (Multi-AZ architectures, Backups, and Snapshots).
  • Weeks 5 - 6 (High Availability & Monitoring): Researched Auto Scaling Groups, Load Balancers, CloudFront distribution, and Route 53 routing. Studied CloudWatch Metrics, Alarms, and Logs Insights dashboards.
  • Weeks 7 - 8 (Automation & Audit Security): Analyzed automation via AWS CLI and automated backup policies using AWS Backup. Evaluated security configurations under AWS S3 SSE-KMS, audited API calls with CloudTrail, and queried logs with Athena SQL.
  • Weeks 9 - 11 (Serverless Operations & Enterprise Networking): Investigated EC2 automation via Lambda and EventBridge triggers integrated with Slack alerts. Researched multi-VPC networking topologies (VPC Peering, Transit Gateway) and hybrid storage with AWS Storage Gateway (NFS).
  • Weeks 12 (Centralized Systems Management & Completion): Studied remote management frameworks via AWS Systems Manager (SSM) (Patch Manager, Run Command) and advanced delegation controls (Permissions Boundaries, Tag-based Access Control) combined with Grafana monitoring.

2. Results in Week 12:

  • Week 12 was dedicated entirely to literature review, conceptual studies, and compiling the final report. No cloud resources were provisioned or modified on the AWS platform.
  • Audited and verified full attendance history and check-ins for the entire course.
