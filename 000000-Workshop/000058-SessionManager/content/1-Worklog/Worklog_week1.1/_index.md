### 1. Session goals
- Understand the role of AWS Budgets in cloud cost control.
- Identify the four main budget types: Cost, Usage, Reserved Instances, and Savings Plans.
- Learn why templates are useful for fast monthly budget setup.

### 2. What was covered
- Reviewed the flow for creating budgets from templates.
- Examined email alert behavior when thresholds are exceeded.
- Noted how AWS forecasts spending to support financial planning.

### 3. Outcomes
- Able to choose a suitable template for a monthly budget.
- Able to distinguish actual cost from forecasted cost.
- Able to describe the purpose of AWS Budgets in day-to-day cloud operations.

> **Note:** This is the most beginner-friendly path because it prioritizes quick setup and immediate visibility into spending.
